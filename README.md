# Dashboard-Xbox-Exel

### Este Dashboard foi feito utilizando a base de dados provida pela DIO e TOTVS para o bootcamp "Fundamentos da Engenharia de Dados e Machine Learning"
A base é uma planilha do exel, contendo dados fictícios de assinaturas do Game Pass, EA Sason Pass e Minecraft Season pass de diferentes clientes.

### O objetivo deste dashboard é servir de ferramenta para monitorar/consutar seguintes perguntas de negócio:
Qual o faturamento total de venda de todos os planos, mês a mês?
Qual o faturamento total de venda, de planos anuais, separado por auto renovação e sem auto renovação, separado mês a mês?
Qual o total de vendas de assinaturas da EA Play?
Qual o total de vendas de assinaturas do Minecraft?

### Na produção deste trabalho, aprendi e apliquei os seguintes conceitos:
Ferramenta exel
Qual a diferença entre dados x informação
Como trabalhar com Big Numbers em uma visualização
Tabelas dinâmicas do exel
Gráficos dinâmicos do exel
Quais as informações necessárias dependendo do alvo do dashboard
Boas práticas ao montar uma visualização

### Para visualizar o dashboard:
Basta clonar este repositório e abrir o documento "dashboard-vendas-xbox-guilherme-vogt" no Exel 365.
Ou baixar o arquivo citado e abrir no Exel 365.
